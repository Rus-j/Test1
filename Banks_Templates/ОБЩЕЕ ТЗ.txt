Нужно сделать разметку PDF документа на чистом PHP с использованием библиотеки PyPDF2 для дальнейшего использования этой разметки как шаблона для генерации PDF документа

За каждым полем разметки закрепляется координата X,Y ; шрифт; размер ; атрибут поля

Координаты X,Y, размеры должны четко совпадать с оригиналом предоставленного документа 

Так же нужно учесть возможность повторяющихся вариативных полей (для транзакций)

В задачу по разметки так же входит размещение по необходимости линий/других форм, таблиц, изображений

Нужно максимально точное позиционирование "пиксель-в-пиксель", шрифты и заготовки (с постоянными данными) будут предоставлены

Подробнее по конкретным шаблнонам внутри соотвествующих папок
